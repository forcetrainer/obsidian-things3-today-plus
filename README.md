# Things3 Today Plus

> Forked from [obsidian-things3-today](https://github.com/wudanyang6/obsidian-things3-today) by wudanyang6

An Obsidian plugin that lets you **view and create** tasks in Things 3's Today list directly from a sidebar pane. macOS only.

## Features

- Displays your Today list from Things 3 within Obsidian
- **Quick-add tasks** to Today via a modal dialog or command palette
- Mark tasks as complete with a single click
- Auto-refreshes every 30 seconds

## Installation

### With BRAT (recommended for beta)

1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat)
2. In BRAT settings, click "Add Beta Plugin"
3. Enter: `forcetrainer/obsidian-things3-today-plus`
4. Enable the plugin

### Manual

1. Download the latest release from [Releases](https://github.com/forcetrainer/obsidian-things3-today-plus/releases)
2. Extract into your vault's `.obsidian/plugins/things3-today-plus/` directory
3. Enable the plugin in Settings > Community Plugins

## Usage

1. Click the check-square icon in the ribbon or use the command palette ("Open Today") to open the sidebar
2. View and complete tasks directly from the pane
3. Click the **+** button or use the command palette ("Add Task to Today") to create a new task
