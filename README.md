# Things 3 Today Plus

> Forked from [obsidian-things3-today](https://github.com/wudanyang6/obsidian-things3-today) by wudanyang6

A powerful Obsidian sidebar plugin for Things 3 on macOS. View your Today list, add new tasks, and jump into Things — all without leaving Obsidian.

## Features

- **Today at a glance** — Your Things 3 Today list rendered in a clean sidebar pane
- **Add tasks from Obsidian** — Create new to-dos with a title, notes, tags, and a flexible date picker (Today, Tomorrow, or any date)
- **One-click complete** — Check off tasks directly from the sidebar
- **Quick links into Things** — Open Today or Upcoming in Things 3 with a single click
- **Auto-refresh** — Stays in sync with Things, refreshing every 30 seconds (plus a manual refresh button)
- **Command palette support** — "Open Today" and "Add Task to Today" available as commands

## Installation

### With BRAT (recommended)

1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat)
2. In BRAT settings, click "Add Beta Plugin"
3. Enter: `forcetrainer/obsidian-things3-today-plus`
4. Enable the plugin

### Manual

1. Download the latest release from [Releases](https://github.com/forcetrainer/obsidian-things3-today-plus/releases)
2. Extract into your vault's `.obsidian/plugins/things3-today-plus/` directory
3. Enable the plugin in Settings > Community Plugins

## Usage

1. Click the check-square icon in the ribbon or run **"Open Today"** from the command palette
2. View and complete tasks directly from the pane
3. Click **"Add To-Do"** to create a new task with notes, tags, and scheduling
4. Click **"Open Things"** to jump to Today in Things 3
5. Click **"Show Upcoming"** to view your upcoming schedule in Things 3

## Requirements

- macOS (uses JXA to communicate with Things 3)
- [Things 3](https://culturedcode.com/things/) installed
